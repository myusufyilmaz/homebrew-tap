# homebrew-tap

Homebrew formulae by [@myusufyilmaz](https://github.com/myusufyilmaz).

## Install

```bash
brew install myusufyilmaz/tap/gatekept
```

Or tap first, then install:

```bash
brew tap myusufyilmaz/tap
brew install gatekept
```

## Formulae

| Formula | Description |
|---|---|
| [`gatekept`](https://github.com/myusufyilmaz/gatekept) | macOS security audit & optimization — catches fake/cracked/tampered apps signature AV misses |
